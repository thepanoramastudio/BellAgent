คู่มือ WePOS ฉบับมาสเตอร์คลาส

ยินดีต้อนรับสู่โลกของการบริหารธุรกิจแบบมือโปรที่ "ความเป๊ะ" คือกุญแจสำคัญสู่ความมั่งคั่งค่ะ คู่มือฉบับนี้ไม่ได้มาเพื่อสอนแค่การใช้เครื่องมือ แต่คือการวางกลยุทธ์ "Command & Control" เพื่อเปลี่ยนทุกยอดขายให้เป็นกำไร (ROI) สูงสุด ด้วยระบบ WePOS ที่ทันสมัยที่สุดในยุคนี้ เตรียมตัวให้พร้อม เพราะเราจะเปลี่ยนร้านคุณให้เป็นระบบระดับสากลแบบตัวแม่ค่ะ!

เล่มที่ 1: WePOS Application สำหรับพนักงานหน้าร้าน (Front-of-House Mastery)

หัวใจของหน้าร้านคือ "ความเร็วและความถูกต้อง" ค่ะ พนักงานของคุณคือด่านหน้าในการสร้างรายได้ ดังนั้นทุกสัมผัสบนแอปพลิเคชันต้องเฉียบคมและแม่นยำ เพื่อความประทับใจของลูกค้าและประสิทธิภาพของระบบหลังบ้านค่ะ

1. การเริ่มต้นใช้งานและการเข้าสู่ระบบ (The Grand Opening)

การเริ่มต้นที่ถูกต้องคือชัยชนะไปกว่าครึ่งค่ะ

* การเข้าสู่ระบบ: ใช้เบอร์โทรศัพท์ที่ลงทะเบียนและรหัสผ่านที่ตั้งไว้ ข้อมูลต้องตรงตามที่ระบุในฐานข้อมูลระบบค่ะ
* การเลือกหน้างาน: ก่อนเริ่มกะ พนักงานต้องเลือก "Project / Brand / Shop" ให้ถูกต้องแม่นยำ เพื่อให้ข้อมูลยอดขายวิ่งเข้าสู่ช่องทางที่ถูกต้อง ไม่ปะปนกันให้เจ้าของร้านปวดหัวภายหลังค่ะ

2. เทคนิคการขายและจัดการบิล (Seamless Sales Workflow)

"ความผิดพลาดเท่ากับศูนย์" คือคติประจำใจของพนักงานระดับมาสเตอร์คลาสค่ะ:

* [ ] ตรวจสอบรายการสินค้า: เช็ค "Product Name" และ "Price" ให้ดีก่อนกดยืนยัน
* [ ] ความเร็วระดับสายฟ้าแลบ: แนะนำให้ใช้ Barcode สแกนสินค้าแทนการกดเลือกด้วยมือในรายการที่มีจำนวนมาก เพื่อลดระยะเวลารอคอยของลูกค้าและเพิ่มความแม่นยำค่ะ
* [ ] การจัดการบิล: พนักงานต้องใช้ฟีเจอร์ "พักบิล" ให้เป็นประโยชน์ในกรณีที่ลูกค้ายังเลือกสินค้าไม่เสร็จ เพื่อให้การบริการลูกค้าท่านอื่นดำเนินต่อไปได้โดยไม่มีสะดุด (Flow of Service) ค่ะ

3. ฟีเจอร์ร้านอาหารและการจัดการโต๊ะ (Professional Dining Setup)

สำหรับธุรกิจร้านอาหาร "Table Turn-over Rate" คือตัวตัดสินกำไรค่ะ:

* จัดการโต๊ะแบบโปร: ดูสถานะโต๊ะได้ทันที และส่งออเดอร์เข้าครัวโดยตรง ลดความผิดพลาดจากการจดมือที่เป็นระบบแบบเก่าค่ะ
* ยืดหยุ่นตามหน้างาน: ระบบรองรับการ "รวมบิล" หรือ "แยกบิล" ได้อย่างรวดเร็ว ตอบโจทย์ไลฟ์สไตล์ลูกค้าสมัยใหม่ ทำให้การปิดโต๊ะทำได้ไวขึ้น พร้อมรับลูกค้ากลุ่มถัดไปทันทีค่ะ

4. การจัดการรอบการขายและปิดยอด (Closing the Shift like a Pro)

ปิดกะให้สวย ยอดเงินต้องตรงเป๊ะเพื่อส่งไม้ต่อให้ระบบ Backend:

* เปิด-ปิดกะ: บันทึกยอดเงินสดตอนเริ่มต้นและจบวันอย่างเคร่งครัด
* Sync ข้อมูล: ระบบจะสรุปยอดขายแยกตามช่องทางชำระเงินโดยอัตโนมัติ เพื่อให้เจ้าของร้านเห็น "Real-time Data" ได้ทันที ความเป๊ะคือที่สุดค่ะ!

เล่มที่ 2: WePOS Web Backend สำหรับเจ้าของร้าน (Executive Operations & Analytics)

นี่คือ "ศูนย์บัญชาการ" (Control Center) ของเจ้าของร้านตัวจริงค่ะ ข้อมูลที่นี่คืออาวุธที่คุณจะใช้ฟันฝ่าคู่แข่งและเพิ่มกำไรให้ธุรกิจอย่างยั่งยืน

1. การเข้าถึงศูนย์บัญชาการ (System Access & Security)

ความปลอดภัยและการเข้าถึงข้อมูลคือรากฐานของความมั่นคงค่ะ

* ทางเข้าระบบ: เข้าใช้งานผ่าน Google Chrome หรือเบอร์เซอร์ที่รองรับที่ URL: https://marketplace.weomni.com
* การ Login: ใช้เบอร์โทรศัพท์ที่ลงทะเบียนไว้ หากใช้งานครั้งแรกให้กด "Forgot your password?" เพื่อตั้งรหัสผ่านใหม่
* จุดสำคัญที่ห้ามพลาด (Technical Nuance): หลังจากเข้าสู่ระบบแล้ว เมื่อมี Popup ให้เลือก Project/Brand/Shop หากคุณมีมากกว่า 1 สาขา ให้เลือก "All" ก่อนกด Select เพื่อดูภาพรวมธุรกิจทั้งหมดในคราวเดียวค่ะ

Security Note: กฎเหล็ก 7 ข้อในการตั้งรหัสผ่าน เพื่อป้องกันผู้ไม่หวังดี ระบบจะบังคับให้คุณตั้งรหัสผ่านโดยต้องผ่านเงื่อนไข "ทั้ง 7 ข้อพร้อมกัน" ปุ่ม Change Password ถึงจะทำงานค่ะ:

1. ต้องมีตัวอักษรพิมพ์เล็ก (a-z)
2. ต้องมีตัวอักษรพิมพ์ใหญ่ (A-Z)
3. ต้องมีตัวเลขอย่างน้อย 1 ตัว
4. ความยาวอย่างน้อย 8 ตัวอักษร
5. ต้องมีอักขระพิเศษหรือเว้นวรรค
6. ห้ามมีช่องว่างที่ขึ้นต้นหรือลงท้าย
7. รหัสผ่านทั้งสองช่องต้องตรงกันเป๊ะค่ะ

2. Dashboard: วิเคราะห์ธุรกิจแบบฉบับตัวแม่ (The ROI Dashboard)

Dashboard คือที่ที่คุณจะมองเห็น "ชีพจร" ของธุรกิจผ่านค่าสถิติ 7 รายการหลักค่ะ

ค่าสถิติ	ความหมายเชิงกลยุทธ์ (Strategic Insight)
Total Sales (Before Discount)	มูลค่าการขายทั้งหมดที่ "ควรจะได้" หากไม่มีโปรโมชั่น
Total Sales (After Discount)	ยอดขายสุทธิที่รับเงินจริง นี่คือตัวเลขเงินสดเข้าเป๋าค่ะ
Discount	ยอดรวมส่วนลด (เปรียบเทียบ Before vs After เพื่อดู Marketing Burn Rate หากส่วนลดสูงเกินไป กำไรคุณจะบางลงนะคะ!)
Total Cancel	ยอดรวมรายการที่ถูกยกเลิก (ใช้ตรวจสอบความโปร่งใสและทุจริตภายในร้านค่ะ)
Tax (VAT)	ภาษีมูลค่าเพิ่มที่เตรียมส่งสรรพากร จัดการให้เป๊ะตามกฎหมายค่ะ
Cost	ต้นทุนสินค้าที่ขายออกไป (COGS)
Profit (กำไร)	(ยอดขายรวม - ต้นทุน) คือตัวเลขความสำเร็จที่ "ตัวแม่" ต้องโฟกัสค่ะ

* Best Seller: ตรวจสอบอันดับสินค้าขายดีเพื่อวางแผนสต็อก (Inventory Planning) ไม่ให้เงินจมไปกับสินค้าที่ขายไม่ออกค่ะ

3. การจัดการสินค้าและสต็อกมหาศาล (Product Management Masterclass)

ระบบ WePOS รองรับโครงสร้างสินค้าที่ซับซ้อนได้อย่างมืออาชีพ:

* Category & Unit: แยกหมวดหมู่ให้ชัดเจนเพื่อรายงานที่อ่านง่าย
* Modifier Group: เพิ่มลูกเล่นและกำไรต่อหน่วย (Upselling) ด้วยตัวเลือกเสริม
* BOM (Bill of Materials): นี่คือ "Invisible Profit Protector" ค่ะ ผูกสูตรอาหารเพื่อตัดสต็อกวัตถุดิบอัตโนมัติ ป้องกันของหายและควบคุมต้นทุนได้อย่างชะงัด
* Product Status Tabs: เลือกดูสถานะสินค้าได้รวดเร็วผ่าน 3 แท็บ (All, Active, Inactive) เพื่อคัดเฉพาะรายการที่พร้อมขายเท่านั้นค่ะ

Focus: การนำเข้าข้อมูลแบบสายฟ้าแลบ (Bulk Import via Excel) หากมีสินค้าเป็นร้อยเป็นพันรายการ ไม่ต้องกดทีละตัวค่ะ ใช้ทางลัด 5 ขั้นตอนดังนี้:

1. เข้าเมนู Import (มุมขวาบน)
2. กด Download Example (ห้ามกรอกไฟล์เองโดยไม่ใช้ Template ระบบนะคะ)
3. จัดเตรียมข้อมูล: กรอกรายละเอียดลงใน Excel (ระบุ SKU และบาร์โค้ดให้ชัดเพื่อความไวหน้าร้าน)
4. Upload: เลือกไฟล์และกด Upload
5. Check: ตรวจสอบสถานะความถูกต้องในระบบ

ฟิลด์ข้อมูลสำคัญใน Excel (ความเป๊ะคือหัวใจค่ะ)

ฟิลด์ข้อมูล	เงื่อนไขความสำคัญ
SKU*	(จำเป็น) รหัสสินค้า ห้ามซ้ำกันเด็ดขาด!
Barcode	เลขบาร์โค้ดสำหรับสแกน (หัวใจของความเร็วหน้าร้านค่ะ)
Product Name*	(จำเป็น) ชื่อสินค้าที่ลูกค้าจะเห็นในใบเสร็จ
Category*	(จำเป็น) หมวดหมู่สินค้าเพื่อความเป็นระเบียบของรายงาน
Price*	(จำเป็น) ราคาขายต่อหน่วย
Custom Price	ใส่ Y หากต้องการให้พนักงานระบุราคาเองได้ / ใส่ N เพื่อขายราคาตายตัว
VAT Calculation*	(จำเป็น) ใส่ Y เพื่อคิดภาษี / ใส่ N หากสินค้านี้ยกเว้นภาษีค่ะ

4. ระบบรายงานเชิงลึก (Deep-Dive Reporting)

รายงานที่เฉียบคมคือที่มาของการตัดสินใจที่ถูกต้องค่ะ:

ประเภทรายงาน	ประโยชน์ต่อธุรกิจ (Masterclass ROI)
Order Summary	ดูภาพรวมยอดขายและพฤติกรรมการซื้อ เพื่อหา Peak Hour ของร้าน
Order Transactions	ตรวจสอบย้อนหลังทุก Transaction เพื่อความโปร่งใส 100%
Tender Report	สรุปวิธีการชำระเงิน ตรวจสอบว่าเงินสดในมือตรงกับระบบหรือไม่
Item Sale Report	วิเคราะห์สินค้า Hero (กำไรดี) และสินค้า Zero (ขายไม่ออก) เพื่อปรับเมนู
Sale VAT Report	รายงานภาษีมูลค่าเพิ่มที่พร้อมส่งฝ่ายบัญชี/สรรพากรทันที
Stock Movement	ตรวจสอบการรับเข้า-จ่ายออก ป้องกันการรั่วไหลของวัตถุดิบ
Stock Balance	รู้ยอดคงเหลือปัจจุบัน เพื่อการสั่งซื้อสินค้าที่แม่นยำ เงินไม่จมค่ะ

ขั้นตอนการ Export Report (เคล็ดลับความเป๊ะ):

1. ไปที่เมนู Report > Export Report
2. เลือก Report Type, ช่วงเวลา, แบรนด์ และ สาขา (สามารถเลือกหลายสาขาพร้อมกันได้ค่ะ)
3. กด Export และรอจน Status เป็น "Success"
4. Master Tip: หากสถานะยังไม่อัปเดต ให้คุณ "กด Refresh บนหน้าเว็บ" หนึ่งครั้ง แล้วปุ่ม Download จะปรากฏขึ้นมาให้คุณจัดการต่อค่ะ!

5. การจัดการหลังบ้านและการลงทะเบียนภาษี (Compliance & Staff Management)

บริหารงานแบบมืออาชีพด้วยความโปร่งใส:

* User Management: กำหนดสิทธิ์พนักงานให้เหมาะสม (Role-based Access Control) เพื่อป้องกัน "รูรั่ว" และความลับทางการค้า เพราะตัวแม่ไม่เคยปล่อยให้กำไรหลุดลอยไปจากการตรวจสอบที่หลวมค่ะ!
* Payment Channels: ตั้งค่าช่องทางการรับเงินให้ครบถ้วน เพื่อความถูกต้องในการคำนวณภาษี (VAT) และความสะดวกของลูกค้าค่ะ
