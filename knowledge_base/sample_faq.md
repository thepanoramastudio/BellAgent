## 3. การสร้างร้านใหม่

### ขั้นตอนการสร้างร้านใหม่

1. ลูกค้ากรอกแบบฟอร์มสร้างร้านเข้ามาในระบบ

   - **User:** 090XXXXXXX (เบอร์โทรที่ลงทะเบียน)
   - **Pin:** 123456 (รหัสเริ่มต้น)
   - วิธีการใช้งานเบื้องต้น: https://weomni.com/wepos-support/getting-started/
   - วันที่เริ่มใช้งานได้

## 4. การเข้าสู่ระบบ WePOS

### วิธีเข้าสู่ระบบ

1. กรอก **เบอร์โทรศัพท์** ที่เป็นบัญชีผู้ใช้
2. กรอก **รหัสพินโค้ด (PIN Code)** 6 หลัก
3. กดปุ่ม "เข้าสู่ระบบ"

### ลืมรหัส PIN / Reset PIN

1. กด **"ลืมรหัส"** ที่หน้าเข้าสู่ระบบ
2. กรอก **เบอร์โทรศัพท์** เพื่อรับ OTP ทาง SMS
3. กรอกเลข OTP ที่ได้รับ แล้วกด "ยืนยัน"
4. ตั้งรหัสพิน **6 หลักใหม่** กรอกสองครั้งให้ตรงกัน แล้วกด "ยืนยัน"
5. ระบบแจ้ง "เปลี่ยนรหัสพินโค้ดสำเร็จ" — เข้าใช้งานได้เลย

### เข้าระบบไม่ได้ / Error "ผู้ใช้หรือรหัสผ่านผิด"

1. ตรวจสอบอินเตอร์เน็ต — ปิด/เปิด Router/WiFi แล้วเชื่อมต่อใหม่
2. ลองใส่รหัสเริ่มต้น **123456**
3. หากยังไม่ได้ ให้กด "ลืมรหัส" เพื่อ Reset PIN ผ่าน OTP

### เข้าระบบแล้วไม่พบข้อมูลร้านสาขา

1. ปิด/เปิด Router/WiFi แล้วเชื่อมต่อใหม่
2. ลองเปลี่ยนเป็นสัญญาณ Hotspot มือถือ
3. เข้าสู่ระบบใหม่อีกครั้ง

## 5. ปัญหาการใช้งาน WePOS (Software)

### แอปค้างที่หน้า Logo / เปิดไม่ได้

1. ปิดเครื่อง POS แล้วเปิดใหม่ (Restart)
2. เปิดแอป WePOS อีกครั้ง
3. หากค้างที่หน้า Logo ให้ **ปัดหน้าจอจากด้านล่างขึ้น** เพื่อหาปุ่มออก แล้วกดออกไปหน้าตั้งค่า

### แอปเด้งออก (Crash)

1. Restart เครื่อง POS
2. เปิดแอป WePOS อีกครั้ง

### ขายสินค้าไม่ได้ / ชำระเงินไม่ได้ (Error)

1. ตรวจสอบอินเตอร์เน็ต — ปิด/เปิด Router/WiFi
2. Restart เครื่อง POS
3. **ไม่แนะนำ Re-Login** เพราะอาจทำให้เข้าไม่ได้

### กดเมนูสินค้าในหน้าขายไม่ได้

1. ปิด/เปิด WiFi และเชื่อมต่อใหม่
2. ลองเปลี่ยนเป็น Hotspot มือถือ
3. เข้าสู่ระบบใหม่

### ซิงค์ข้อมูลไม่สำเร็จ / ปิดรอบการขายไม่ได้

1. ตรวจสอบอินเตอร์เน็ต
2. กดปุ่ม **Sync** อีกครั้ง (วิธีกด Sync: https://tinyurl.com/Press-sync)

### ราคาสินค้าเปลี่ยนเอง / สต็อกไม่ถูกต้อง

กดปุ่ม **Sync** ให้ซิงค์ข้อมูลเสร็จสมบูรณ์: https://tinyurl.com/Press-sync

### เปิด/ปิดรอบการขายไม่ได้

1. ตรวจสอบอินเตอร์เน็ต — ปิด/เปิด Router/WiFi
2. ลองเปลี่ยนเป็น Hotspot มือถือ

## 6. คู่มือการใช้งาน WePOS (ฟีเจอร์หลัก)

### วิธีเพิ่มสินค้าใหม่ / สร้างสินค้าใหม่

1. กดเมนูหลัก (ไอคอนเส้น 3 ขีด)
2. เลือก **จัดการสินค้า**
3. เลือก **สินค้า**
4. กด **เพิ่มสินค้า**
5. กรอกชื่อสินค้า และเลือกหมวดหมู่สินค้า
6. กรณีสินค้าไม่มีตัวเลือก: เลือก "ไม่มีตัวเลือก" แล้วกรอกราคาขายและบาร์โค้ด
7. กรณีสินค้ามีตัวเลือก (เช่น ขนาด สี): เลือก "มีตัวเลือก" แล้วกรอกรายละเอียดตัวเลือกย่อย
8. กด **บันทึก**

> กรณีต้องการกำหนดราคาขณะขาย: เปิดฟังก์ชั่น "ระบุราคาเมื่อขาย" ก่อนบันทึก

### วิธีเพิ่มหมวดหมู่สินค้า

1. กดเมนูหลัก
2. เลือก **จัดการสินค้า**
3. เลือก **หมวดหมู่สินค้า**
4. กด **เพิ่มหมวดหมู่สินค้า**
5. กรอกชื่อหมวดหมู่
6. กด **บันทึก**

### วิธีแก้ไขข้อมูลสินค้า

1. กดเมนูหลัก → จัดการสินค้า
2. พิมพ์หรือสแกนบาร์โค้ดสินค้าที่ต้องการแก้ไขในช่องค้นหา
3. กดรูปดินสอเพื่อแก้ไข
4. แก้ไขข้อมูลตามต้องการ แล้วกด **บันทึก**

### วิธีขายสินค้า

1. กดเลือกสินค้าจากหมวดหมู่ หรือใช้สแกนเนอร์สแกนบาร์โค้ด
2. กด **+** หรือ **-** เพื่อปรับจำนวน
3. ตรวจสอบรายการ แล้วกด **ชำระเงิน**
4. เลือกวิธีชำระเงิน (เงินสด / QR / TMN)
5. กด **ยืนยันการชำระเงิน**

### วิธีพักบิล และเรียกบิล

**พักบิล:**
1. เลือกสินค้าแล้วกด **พักบิล**
2. ตั้งชื่อบิล แล้วกด **ตกลง**

**เรียกบิลที่พัก:**
1. กด **เรียกบิล**
2. เลือกรายการบิลที่ต้องการ
3. กด **ชำระเงิน**

### วิธีเปิด/ปิดรอบการขาย

**เปิดรอบการขาย:**
1. กดเมนูหลัก → จัดการเงินสด
2. กรอกยอดเงินสดเริ่มต้น (เงินทอน)
3. กด **เปิดรอบการขาย**

**ปิดรอบการขาย:**
1. กดเมนูหลัก → จัดการเงินสด
2. กด **ปิดรอบการขาย**
3. ใส่จำนวนเงินสดที่ขายได้ (รวมเงินทอนเริ่มต้น)
4. กด **ยืนยัน**

### วิธีดูรายงานยอดขาย

1. กดเมนูหลัก → **รายงาน**
2. เลือกวันที่ที่ต้องการ (ดูได้สูงสุด 31 วัน)
3. ดูสรุปยอดขาย, รายการสินค้า, ยอด VAT, ต้นทุน, กำไรได้ในหน้าเดียว

### วิธีเพิ่มโปรโมชั่น

1. กดเมนูหลัก → **จัดการโปรโมชั่น**
2. กด **เพิ่มโปรโมชั่น**
3. กรอกชื่อ, โค้ด, คำอธิบาย, ระยะเวลาโปรโมชั่น
4. เลือกรูปแบบส่วนลด: (A) ลดท้ายบิล (B) เฉพาะสินค้าราคาต่ำสุด (C) สินค้าที่ร่วมรายการ
5. ใส่จำนวนส่วนลด (บาท หรือ %) และจำนวนคูปอง
6. กด **บันทึก**

### วิธีจัดการสต็อกสินค้า

1. กดเมนูหลัก → **จัดการสต็อก**
2. ค้นหาสินค้าด้วยชื่อหรือบาร์โค้ด
3. กด **+** หรือ **-** หรือรูปดินสอ
4. ใส่จำนวนสินค้า เลือกเหตุผล แล้วกด **บันทึก**

### วิธียกเลิกใบเสร็จ

1. กดเมนูหลัก → **ใบเสร็จ**
2. เลือกวันที่ แล้วเลือกใบเสร็จที่ต้องการยกเลิก
3. กด **ยกเลิกใบเสร็จ** → เลือกเหตุผล → กด **ยืนยัน**

### วิธีพิมพ์สำเนาใบเสร็จ / ออกใบกำกับภาษี

1. กดเมนูหลัก → **ใบเสร็จ**
2. เลือกวันที่ แล้วเลือกรายการที่ต้องการ
3. กด **พิมพ์ใบเสร็จ**
4. เลือกประเภท: ใบเสร็จแบบย่อ หรือ **ออกใบกำกับภาษี** (IV)

> ออกใบกำกับภาษีได้เฉพาะวันที่ทำรายการเท่านั้น (ข้ามวันไม่ได้)
> ต้องเปิดฟังก์ชั่น VAT ใน เมนู → ตั้งค่า → รายละเอียดร้านค้า → การจัดการภาษีมูลค่าเพิ่ม ก่อน

### วิธีสร้างบัญชีพนักงาน

1. เข้าสู่ระบบด้วยบัญชีเจ้าของร้าน
2. กดเมนูหลัก → **จัดการพนักงาน**
3. กด **เพิ่มพนักงาน**
4. กรอกชื่อ / เลือกตำแหน่ง / กรอกอีเมล / เบอร์โทร / รหัส PIN 6 หลัก
5. กด **บันทึก**

### วิธีเปลี่ยนโลโก้บนใบเสร็จ

1. นำรูปภาพเข้าเครื่อง POS ก่อน
2. กดเมนูหลัก → ตั้งค่า → เครื่องพิมพ์ → รูปแบบใบเสร็จ
3. กดไอคอนกล้อง → เลือกภาพ
4. กด **บันทึก** แล้วกด **Sync**

### เครื่องพิมพ์ไม่เชื่อมต่อ / ลิ้นชักไม่เด้ง

1. ออกจากระบบและปิดเครื่อง
2. เปิดฝาเครื่องพิมพ์ นำกระดาษออกแล้วใส่กลับ (ดึงปลายออกประมาณ 15 ซม. ก่อนปิดฝา)
3. เปิดเครื่องใหม่ — เครื่องจะ detect กระดาษและ printer เอง
4. เปิด WePOS → เมนู → ตั้งค่า → เครื่องพิมพ์ → เครื่องพิมพ์แคชเชียร์ → USB → เลือกรุ่น (iMIN D4 / SUNMI) → บันทึก

### ใช้งานแบบออฟไลน์ได้หรือไม่?

ได้ แต่มีข้อจำกัด:
- **ทำได้:** ขายสินค้า ให้ส่วนลดท้ายบิล ชำระเงินสด
- **ทำไม่ได้:** เข้า/ออกจากระบบ, พักบิล, โปรโมชั่น, ชำระผ่าน TMN/QR, จัดการสินค้า/ใบเสร็จ/รอบการขาย, คลัง, รายงาน, ตั้งค่า
- เมื่อกลับมาออนไลน์ ระบบจะ Sync ข้อมูลการขายอัตโนมัติ

## 8. การรับประกันและบริการหลังการขาย

### ประเภทการรับประกัน

| ประเภท | รายละเอียด |
| --- | --- |
| **Standard Warranty (1 ปี)** | 7 วันแรก: เปลี่ยนเครื่องใหม่ / หลัง 7 วัน: Pickup-Repair-Return หรือ Walk-in ที่ศูนย์บริการ |
| **On-site Device Repair (1-3 ปี)** | ช่างเข้าซ่อมที่ร้านลูกค้า (ไม่มีเครื่องสำรอง) |
| **On-site Device Swap (1-3 ปี)** | ช่างเข้าเปลี่ยนเครื่องทดแทนที่ร้านลูกค้า |

### ระยะเวลาดำเนินการ (Standard Warranty)

- **Pickup-Repair-Return:** นัดรับภายใน 3 วันทำการ → ซ่อม 3-5 วันทำการ → ส่งกลับ (กทม. 1 วัน, ตจว. 2 วัน)
- **Walk-in:** ลูกค้าเดินเข้าศูนย์บริการ → ซ่อม 3-5 วันทำการ
- **In Warranty:** ไม่มีค่าใช้จ่าย
- **CID (ความเสียหายที่ผู้ใช้เป็นต้นเหตุ):** มีค่าอะไหล่ตามจริง
- **End of Warranty:** ลูกค้าตกลงกับศูนย์ซ่อมโดยตรง

### ระยะเวลาดำเนินการ (On-site Repair/Swap)

### สิทธิ์พนักงาน 2 ระดับ
