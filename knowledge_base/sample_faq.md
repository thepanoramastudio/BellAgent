# WePOS General FAQ

## วิธีติดต่อ Support
- LINE @wepos: 07:00–22:00 ทุกวัน
- Call Center: 02-0202364 กด 1 (09:00–18:00 ทุกวัน)

## ระบบ Maintenance
Scheduled maintenance ทุกวันอาทิตย์ 00:00–04:00 น. (เวลาประเทศไทย)

## การสมัคร Payment Package
ไปที่ Settings → Payment → Subscription แล้วเลือก Package ที่ต้องการ

## วิธี Reset Password
ติดต่อ Admin เพื่อ Reset Password ผ่านเมนู Settings → Users
