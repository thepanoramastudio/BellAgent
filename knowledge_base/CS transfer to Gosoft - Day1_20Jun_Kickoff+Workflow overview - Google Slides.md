## หน้า 33

Workflow and SOPs
● ส่ง request ขอ Onboarding ร้านค้าใหม่
○ CPF
i. รับ Request >> CPF Order Form
ii. เปิด jira ticket ให้ L2 >> how to input ข้อมูล
iii. ส่งขอสร้างร้านกับ SO >> WePOS new shop creation request form เลือก company = บริษัท ซีพีเอฟ เรสเทอรองท แอนด ฟูด
เชน จำกัด
iv. ได้รับ master product จาก CPF admin ผ่าน Line OA (K มีน)
v. CS upload master product เข้าไปในร้าน CPF ผ่าน Backend
vi. เก็บ Record ของ CPF >> CPF Fivestar-staff-Import_2022-2025
○ MASS (Marketplace)
i. เห็น order จาก chat
ii. ถามรับข้อมูลจาก Chat (template ข้อมูล) ใน Marketplace
iii. เปิด jira ticket ให้ L2 >> how to input ข้อมูล
iv. ส่งขอสร้างร้าน + product กับ SO >> WePOS new shop creation request form เลือก company = MASS
v. ถ้ามีติดตั้ง ส่งแบบ form Confirm installation ให้ลูกค้ากรอกแบบ form และโอนเงิน
vi. Check ใน form response ของ form Confirm installation ถ้ามีเงินเข้าแจ้ง finance ทาง email
vii. Assign งานให้กับ Vendor ติดตั้ง Eak&W >> Eak&W request form หรือ Ditto >> Ditto request form
viii. รับเรื่องประสานงานจาก Vendor ทาง Line group > AscendxEak&W POS Support, AscendxDitto POS Support

## หน้า 34

Workflow and SOPs
○ MASS (Sale)
i. Sale ส่ง order ให้ทาง email (Weomni-cs@ascendcorp.com)
ii. เปิด jira ticket ให้ L2 >> how to input ข้อมูล
iii. ส่งขอสร้างร้าน + product กับ SO >> WePOS new shop creation request form เลือก company = MASS
iv. ถ้ามีงานติดตั้ง Assign งานให้กับ Vendor ติดตั้ง Eak&W >> Eak&W request form หรือ Ditto >> Ditto request form
v. รับเรื่องประสานงานจาก Vendor ทาง Line group > AscendxEak&W POS Support, AscendxDitto POS Support
○ MASS (ไจแอนท์ลูกชิ้นปลา, Ramenga, ร้านค้า WePOS ปัจจุบัน) ต้องการเปิดสาขาหรือร้านค้าเพิ่มเติม
i. L1 ส่งอีเมล์แจ้งความต้องการให้ Sale ส่งใบเสนอราคาให้ลูกค้า
ii. Sale ส่ง order ให้ทาง email (Weomni-cs@ascendcorp.com)
iii. เปิด jira ticket ให้ L2 >> how to input ข้อมูล
iv. ส่งขอสร้างร้าน + product กับ SO >> WePOS new shop creation request form เลือก company = MASS
v. ถ้ามีงานติดตั้ง Assign งานให้กับ Vendor ติดตั้ง Eak&W >> Eak&W request form หรือ Ditto >> Ditto request form
vi. รับเรื่องประสานงานจาก Vendor ทาง Line group > AscendxEak&W POS Support, AscendxDitto POS Support
○ Makro
i. รับข้อมูลจาก sheet >> Makro Setup plan
ii. ไม่ต้องเปิด Jira ticket
iii. เช็คกับทาง Makro ว่าสร้างร้านค้าแล้ว >> Line group MakroPOS x Ascend Support
iv. Assign งานให้กับ Vendor ติดตั้ง Eak&W >> Eak&W request form หรือ Ditto >> Ditto request form
v. รับเรื่องประสานงานจาก Vendor ทาง Line group > AscendxEak&W POS Support, AscendxDitto POS Support
